# 新聞事件三層次解構器

## 核心理念

巨人傑的三層次分析框架：
- **第一層**：看到什麼（表面現象）
- **第二層**：背後是什麼（隱藏資訊）
- **第三層**：我該怎麼做（系統決策）

> 「散戶看新聞做多空，專業交易者看新聞找機會成本。」— 巨人傑

## 分析流程

### 第一層：表面資訊萃取

回答以下問題：
1. **事件本身**：發生了什麼事？（用一句話描述）
2. **關鍵數據**：具體數字是什麼？與預期/前值相比如何？
3. **市場初反應**：消息發布後，相關標的如何反應？
4. **時間軸**：這是已發生、正在發生、還是預期將發生的事？

輸出格式：
```
📰 事件：[一句話描述]
📊 數據：[實際值] vs [預期值] vs [前值]
📈 初反應：[標的] [漲/跌] [幅度]
⏰ 時間：[已發生/進行中/預期]
```

### 第二層：隱藏資訊挖掘

回答以下問題：
1. **誰是受益者？** 這個事件對誰最有利？
2. **誰是受害者？** 這個事件對誰最不利？
3. **市場定價了嗎？** 這個消息是否已經反映在價格中？
4. **真實動機？** 決策者（央行/公司/政府）的真正意圖是什麼？
5. **市場盲點？** 大多數人還沒意識到什麼？

關鍵思考框架：
- **公開資訊 vs 隱含資訊**：新聞說的 vs 沒說的
- **短期反應 vs 長期影響**：今天漲跌 vs 趨勢改變
- **主流觀點 vs 逆向思考**：大家都這樣想 vs 如果反過來呢

輸出格式：
```
🎯 受益者：[誰] — [原因]
⚠️ 受害者：[誰] — [原因]
💰 定價程度：[未定價/部分定價/充分定價]
🧠 真實動機：[分析]
👁️ 市場盲點：[大多數人忽略的點]
```

### 第三層：系統性決策

回答以下問題：
1. **對我目前部位的影響？** 利多/利空/中性
2. **期望值分析**：如果我要操作，勝率和賠率如何？
3. **時機判斷**：現在行動 vs 等待更好時機
4. **風險評估**：最壞情況是什麼？我能承受嗎？
5. **具體行動**：買/賣/持有/觀望？倉位多少？

巨人傑決策優先順序：
```
守住獲利 > 降低成本 > 賺取額外收益
```

輸出格式：
```
📍 部位影響：[利多/利空/中性] — [原因]
📐 期望值：勝率 [X]% × 獲利 [Y] - 敗率 [Z]% × 虧損 [W] = [EV]
⏳ 時機建議：[立即行動/等待/觀望]
🛡️ 風險評估：最壞情況 [描述]，可承受度 [高/中/低]

🎬 巨人傑建議：
[具體行動方案，包含倉位建議]
```

## 特殊事件處理指引

### 央行決策（Fed/ECB/BOJ）
- 關注點：利率決議 vs 點陣圖 vs 聲明措辭
- 隱藏資訊：鴿派/鷹派程度變化、經濟預測修正
- 常見盲點：市場過度關注利率本身，忽略前瞻指引

### 經濟數據（CPI/非農/GDP）
- 關注點：實際值 vs 預期值 vs 前值修正
- 隱藏資訊：細項數據、趨勢變化、季節性因素
- 常見盲點：headline 數據 vs core 數據的差異

### 財報發布
- 關注點：EPS vs 營收 vs 指引
- 隱藏資訊：毛利率變化、庫存水位、資本支出計畫
- 常見盲點：本季數字 vs 下季展望的權重

### 地緣政治
- 關注點：事件本身 vs 市場反應
- 隱藏資訊：各方利益、談判籌碼、歷史模式
- 常見盲點：短期恐慌 vs 長期影響的差異
